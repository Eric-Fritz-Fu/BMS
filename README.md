# 银行管理系统

## 功能

- 用户
  - 创建用户
  - 用户余额查询
  - 用户登录
  -  ( 管理员 ) 删除用户
  -  ( 管理员 ) 更改用户信息
- 账单
  - 账单查询
  - 转账
  -  ( 管理员 ) 存款/提现

## 数据库

数据库使用自己设计的文件存储, 像 `database` 文件。

第一行两个数, 分别表示用户数 $m$ 和记录数 $n$.
后 $n$ 行, 每行由 $1$ 个数和 $3$ 个字符串组成, 分别表示用户标识号, 用户名, 姓名, 密码 ( $6$ 位 ) .
后 $m$ 行, 每行由 $3$ 个数组成, 分别表示用户账单的发起人, 收款人, 金额.
最后保留一个空行。
